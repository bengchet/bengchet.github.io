Learning Hugo for First Time

Apr 4, 2018 02:56 · 98 words · 1 minute read hugo website

Why I choose Hugo now?

Have been doing some static pages website for my friends and companies, I found that sometimes it is troublesome to commit changes demanded by customers. Without proper templating and content management system (CMS), it can be a headache that I need to go through html pages one by one and edit the content. Seeing the potential what Hugo can do, I have decided to use Hugo as generator for static pages website in the future. The content will be easier to manage without interfering with html tags of pages. Looking forward to it!
